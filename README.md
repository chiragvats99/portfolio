# Chirag Vats Portfolio

Dependency-free portfolio website built from the provided resume, Oracle
experience notes, and supplied photos.

## Run Locally

```bash
npm run dev
```

Then open `http://localhost:5173`.

## Validate

```bash
npm run build
```

The build script verifies that the static entry point, stylesheet, script, images,
and resume PDF are present.

## Source Files

- `index.html` - page content and sections
- `styles.css` - responsive visual system and animations
- `script.js` - pointer parallax and scroll reveal behavior
- `public/images/` - optimized portfolio photos
- `public/chirag-vats-resume.pdf` - linked resume
